## 🌟 Simplify Your Wealth Management
**TR Money** is a premium, high-performance financial management application built with **React Native** and **Expo**. Designed for individuals who value precision and elegance, it offers a seamless way to track expenses, manage multiple accounts, and gain deep insights into your spending habits.

---

## 🚀 Key Features

- 💎 **Modern Glassmorphism UI**: A stunning, premium interface with smooth animations and interactive components.
- 📊 **Dynamic Data Visualization**: Beautifully crafted pie and bar charts powered by `react-native-gifted-charts` for clear spending analysis.
- 🌍 **Localized Experience**: Full support for **English, Thai, Chinese, and Japanese**, ensuring a natural experience across the globe.
- 💱 **Multi-Currency Support**: Effortlessly manage transactions in different currencies with real-time context management.
- 🔒 **Offline-First with Cloud Sync**: Powered by **SQLite** for lightning-fast local storage and **Supabase** for secure cloud synchronization and authentication.
- 🎨 **Theme Customization**: Responsive dark and light modes with custom accent colors designed to be easy on the eyes.
- 🌗 **Responsive Design**: Auto-scaling UI that looks perfect on every screen size, from small phones to large tablets.

## ⚙️ Installation (For Developers)

To run this project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/tr-money-mobile.git
   cd tr-money-mobile
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up Environment Variables**:
   Create a `.env` file in the root directory and add your Supabase credentials:
   ```env
   EXPO_PUBLIC_SUPABASE_URL=YOUR_SUPABASE_URL
   EXPO_PUBLIC_SUPABASE_ANON_KEY=YOUR_SUPABASE_ANON_KEY
   ```

4. **Start the application**:
   ```bash
   npx expo start
   ```
